# Websokets chat 

# Ссылка на проект:

## Чат реализован на Node.JS с использованием фреймврока koa2 и websockets

### В чат можно вставлять любую разметку, это не баг, а фича!

## Build setup

``` bash

# install dependencies

npm install

# run server

npm run dev

# or

node index

```
